Software Engineering

Overview

I’ve written software for a variety of platforms using many different programming languages and toolkits. Before learning programming formally, I dabbled with web development, self-teaching the classic paradigms, tools, and languages for simple sites. In CS courseware and research, I wrote a heap of software in C++, Python and R.


Rigorous SWE projects from courseware accompany my professional work:

  • Image Compression & Decompression
    A program that takes portable pixmap format (PPM) images and compresses them by transforming color spaces and discarding information not easily seen by the human eye.
  • Android Malware Analysis
    Reverse engineering an Android app to determine location and purpose of live malware.
  • Analysis of AMD64 Assembly Code
    Analyzed and debugged C procedures given in executable binary to prevent explosion of a "binary bomb" during runtime.
  • Incident Alarm
    A rudimentary Security Information and Event Management (SIEM) / intrusion detection system (IDS).
  • A Universal Machine
    Software implementation of a simple virtual machine.
  • Simlines
    A program that reads any number of files and detects lines in files that are similar to each other.
  • Unblackedges
    A program that removes black edges from a scanned image.
  • Sudoku Solution Identifier
    Testing for solutions of 9x9 sudoku puzzles via abstractions of 2-D, polymorphic, unboxed arrays.

While building software, I’ve picked up the ability to use a mix of helpful third-party APIs, database technologies and query languages, scripting languages, version control systems, and build tools to aid in the process.
