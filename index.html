Automating Lasercom Terminal Camera Calibration

Ensuring that infrared cameras precisely capture data sent via laser, under any atmospheric condition, is accomplished by running a series of hardware configuration tests. This process takes 2-3 weeks per lasercom terminal. In order to speed up this process, myself and a team of 3 other interns at MIT Lincoln Laboratory built a system that automates this, allowing researchers to run a test suite overnight, and from afar.
